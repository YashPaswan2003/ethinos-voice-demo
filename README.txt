ETHINOS AI VOICE DEMO
=====================
Powered by Sarvam AI APIs

HOW TO RUN
----------
1. Make sure Python is installed on your computer
2. Double-click START.bat (Windows)
   OR run in terminal: pip install flask flask-cors requests && python server.py
3. Open http://localhost:5000 in Chrome browser
4. That's it - both demos are ready to use

WHAT'S INSIDE
-------------
- AI Customer Service Voice (Text-to-Speech)
  Type any text, pick a voice + language, hear the AI speak it

- Speech-to-Text (Hindi/English)
  Click mic, speak, see transcription appear
  Also supports audio file upload

NOTES
-----
- Chrome works best (needs mic access for STT)
- Internet connection required (calls Sarvam AI cloud APIs)
- Supports 10 Indian languages + English
